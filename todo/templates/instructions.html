Welcome to Aledade's code test!

The purpose of this exercise is to test your ability to quickly and iteratively build features in a new code base.

Despite its contrived simplicity, this application uses the same frameworks we utilize in our production environment.

Instructions

Within the time allotted, your task is to:

  1. Execute as many feature stories as possible
  2. Commit your work as you accomplish tasks
  3. Bundle your code (git bundle create /tmp/todo_work.bundle master), then send it to Aledade Tech Recruiting.

The application

You will be building a simple "TODO" application. It already includes a number of tasks, which are themselves the stories you must work on.

You can view stories within the application itself, or as Markdown files in starting_tasks/.
